# Z-оценка (Z-Score) и её применение с SMA и VWAP

## Что такое Z-оценка

Z-оценка (z-score) — это статистическая мера, которая показывает, на сколько стандартных отклонений значение отклоняется от среднего значения. Формула:

```
z = (x - μ) / σ
```

где:
- x — текущее значение
- μ — среднее значение
- σ — стандартное отклонение

## Z-оценка относительно SMA

Когда z-оценка вычисляется относительно SMA (Simple Moving Average), она показывает, на сколько стандартных отклонений текущая цена отклоняется от скользящего среднего:

```
z_score_sma = (price - sma) / std_dev
```

где:
- price — текущая цена
- sma — значение простого скользящего среднего
- std_dev — стандартное отклонение цен за определённый период

## Z-оценка относительно VWAP

VWAP (Volume Weighted Average Price) — это объемно-взвешенное среднее значение цены, которое учитывает объемы торгов. Если использовать VWAP вместо SMA в z-оценке:

```
z_score_vwap = (price - vwap) / std_dev
```

где:
- price — текущая цена
- vwap — значение объемно-взвешенного среднего
- std_dev — стандартное отклонение (может быть вычислено с учетом объемов или без)

## Различия между Z-Score на основе SMA и VWAP

| Показатель | SMA | VWAP |
|------------|-----|------|
| Вес | Равный для всех точек | Больший вес у точек с высоким объемом |
| Реакция на объемы | Не учитывает | Учитывает объемы торгов |
| Применение | Общее сглаживание | Внутридневной анализ с учетом ликвидности |

## Практическое применение

Z-оценка на основе VWAP может быть особенно полезна для:
- Внутридневного трейдинга
- Оценки перекупленности/перепроданности с учетом объемов
- Сравнения текущей цены с "реальным" средним, взвешенным по объему
